Events: Calendar View
This module will provide a calender view for events. With a calendar view, customers can view events for a specific date. All events link to be shown in pop-up for each day.

Features:

  • This module will provide a calender view for events.
  • With a calendar view, customers can view events for a specific date.
  • All events link to be shown in pop-up for each day.

  • Work Flow

Work Flow

Events Calendar View

All events link to be shown in pop-up for each day.
